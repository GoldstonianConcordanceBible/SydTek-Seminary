# Purpose

This standard defines how SydTek Seminary issues, verifies, and manages digital ministry credentials.

Digital credentials support transparency, portability, portfolio-based review, and public verification.

---

# Credential Types

SydTek Seminary may issue:

- Digital Certificates
- Digital Badges
- Ministry Completion Credentials
- Licensed Minister Credentials
- Ordained Minister Credentials
- Senior Minister Credentials
- Optional On-Chain Credentials

---

# Required Credential Fields

Each credential should include:

- Recipient Name
- Credential Title
- Issuing Body
- Issue Date
- Credential ID
- Credential Level
- Verification Status
- Portfolio Evidence Link
